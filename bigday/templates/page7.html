Exceptions handling:-

What are exceptions in Python?
Exceptions are errors that are detected during execution of the code. They are not always fatal for the program. Exceptions can be handled in python for proper execution of the program and get the expected output.

Example: ZeroDivideError , NameError, TypeError etc. The name of the error is displayed in the message followed by the reason for its arrival.
>>> print(10/0)

>>> print(place)

>>> print('India'+1)

Catching Exceptions in Python

Once an error occurs you need to write a block of code for catching and handling it. So, that the program doesn’t crash and gives us the required output.
In python we have a special keyword for catching exceptions if they arise in the ‘try block’.
We write the code that we suspect could result into error in the ‘try block’.
The ‘except block’ is called when the ‘try block’ detects an error. The ‘except block’ then handles the error and it is unto us what we want to do in the ‘except-block’.

Example:
import sys
list=[100,200,300]
no=[0,1,2]
l=len(list)
for i in range (0,l):
    try:
        values=(list[i])/no[i]
    except:
        values=(list[i])/(no[i]+1)
        print('The item was divided by zero')
    print('values=',values)

Output:-
The item was divided by zero
values= 100.0
values= 200.0
values= 150.0

In this program we are basically trying to find the result of division of two list with their consecutive elements.
Hence, when we try to divide the code with the first element of ‘no’ list which is 0 the try block catches the ZeroDivideError and calls the except block to handle it. In the except block we raise the value of it by 1 and the program executes without crashing.


Catching Specific Exceptions in Python
In the above example, we did not mention any specific exception in the except clause.
We should always write different except blocks for different exceptions because all the exceptions have different causes of arousal and must be handled differently.
A try clause can have any number of except clause to handle them differently but only one will be executed in case an exception occurs.
We can use a tuple of values to specify multiple exceptions in an except clause. Here is an example pseudo code.

Example:
try:
   # do something
except ValueError:
   # handle ValueError exception
except (TypeError, ZeroDivisionError):
   # handle multiple exceptions
   # TypeError and ZeroDivisionError
except:
   # handle all other exceptions



Raising Exceptions
If we want to raise an exception or call the except block in the try block, we can use raise keyword.
Raise calls the except block . We can also pass a value with the raise function which can later be used in the except block.

Example:
list=[10,20,30]
try:
    a=int(input('Enter the number to divide all elements of the list:-'))
    if int(a)==0:
        raise ZeroDivisionError('The number entered was zero')
    elif int(a)<0:
        raise ValueError('The number enetered was less than zero')
    else:
        for i in list:
            print('The elements are:-',i/a)
except ZeroDivisionError as o:
    print(o)
except ValueError as o:
    print(o)
