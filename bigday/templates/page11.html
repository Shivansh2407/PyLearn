A Sorting Algorithm is used to rearrange a given array or list elements according to a comparison operator on the elements. The comparison operator is used to decide the new order of element in the respective data structure.

There are many types of sorting algorithms but here we will focus in two of them namely:
 1.Bubble sort
 2.Insertion sort

                                                                            Bubble sort
Bubble Sort is the simplest sorting algorithm that works by repeatedly swapping the adjacent elements if they are in wrong order according to the operation that needs to be performed.

Example:
Consider we have a list:
                                                                             [ 5 , 4 , 6 ,  3 , 1  ]
We have to arrange it in ascending order. So, what we will do is:
1. Compare every element with itâ€™s adjacent element .
                Here, 5>4 so we need to swap.

2. If the condition is not met then, we need to swap them.
We need to arrange in ascending order so, 4 must arrive before 5 in the list. Hence, We swap the two elements. The array now becomes:
                                                                              4 5 6 3 1

3. We continue to check the array and repeat the above steps until the second last element. So, the array we be:
                5<6: No swapping. Array remains the same.
                6>3: Swapping required. Array becomes: 4 5 3 6 1
                6>1:Swapping required. Array becomes: 4 5 3 1 6

 4. Again we need to repeat the above steps (length(list)-1)times to get the final sorted array. The above was the first iteration.

5. Next iteration (2nd). Now the array becomes:
               4<5:No swapping . Array remains same.
               5>3: swapping required. Array becomes: 4 3 5 1 6
               5>1: swapping required. Array becomes: 4 3 1 5 6
               5<6: no swapping .

6. Next iteration (3rd).Now the array becomes:
                4>3: Swapping required. Array becomes: 3 4 1 5 6
                4>1:swapping required. Array becomes: 3 1 4 5 6
                4<5: no swapping.
                5<6: no swapping.

7.Next iteration(4th )(final).Now the array becomes:
                3>1:swapping required. Array becomes: 1 3 4 5 6
                3<4: no swapping.
                4<5: no swapping.
                5<6: no swapping.

Now the 'length(list)-1' iterations are over and the array that we have obtained:
                                                                      [ 1 , 3 , 4 , 5 , 6 ]
The obtained array is sorted in ascending order using the bubble sort algorithm.

Implementation of Bubble sort in python:
This is implemented using two nested for loops and every adjacent element is compared .

# Python program for implementation of Bubble Sort

def bubbleSort(arr):
    n = len(arr)
# Traverse through all array elements
    for i in range(n):
        # Last i elements are already in place
         for j in range(0, n-i-1):
                   # traverse the array from 0 to n-i-1
                   # Swap if the element found is greater
                   # than the next element
                    if arr[j] > arr[j+1] :
                                   arr[j], arr[j+1] = arr[j+1], arr[j]

Complexity and Efficiency
Worst and Average Case Time Complexity: O(n*n). Worst case occurs when array is reverse sorted.
Best Case Time Complexity: O(n). Best case occurs when array is already sorted.
Auxiliary Space: O(1)
Boundary Cases: Bubble sort takes minimum time (Order of n) when elements are already sorted.
Sorting In Place: Yes
Stable: Yes

                                                                          Insertion sort

Insertion sort is a another simple sorting algorithm. This algorithm is based on the addition of elements in a particular order to sort the list.
Algorithm:
The elements need to sorted in ascending order. To do that we need to :
1. Compare every element with the elements previous to it.
2. If the element is lesser than an element prior to it in the list.
3. Then ,Insert that element before the other element .

Example:
Consider a list:
                                                                       [  4 ,3 , 2 , 10 , 12 , 1 5 , 6  ]
 

All Elements in the list:
    4:   no elements before it.
    3:   4>3 so 3 must be inserted before 4 in the list .Hence, the list becomes:
                                                                  3 4 2 10 12 1 5 6
    2:   3>2 so 2 must be inserted before 3. List becomes:
                                                                  2 3 4 10 12 1 5 6
    10: no chances
    12: no changes
     1:   2>1 so 1 must be inserted before 2 in the list. List becomes:
                                                                   1 2 3 4 10 12 5 6
     5:  1<5 :no change
           2<5 :no change
           3<5 : no change
           4<5 : no change
           10>5 : so 5 must be inserted before 10. List becomes:
                                                                      1 2 3 4 5 10 12 6
     6:    1<6 :no change
             2<6 :no change
             3<6: no change
             4<6 : no change
             5<6: no change
             10>6: so 6 must be inserted before 10. List becomes:
                                                                        1 2 3 4 5 6 10 12
List has been sorted!


Implementation of insertion sort in python:

# Python program for implementation of Insertion Sort
# Function to do insertion sort
def insertionSort(arr):
# Traverse through 1 to len(arr)
    for i in range(1, len(arr)):
        key = arr[i]
# Move elements of arr[0..i-1], that are
# greater than key, to one position ahead
# of their current position
        j = i-1
        while j >= 0 and key < arr[j] :
            arr[j + 1] = arr[j]
            j -= 1
            arr[j + 1] = key
# Driver code to test above
arr = [12, 11, 13, 5, 6]
insertionSort(arr)
for i in range(len(arr)):
    print ("% d" % arr[i])

Output:
5 6 11 12 13

Complexity and Efficiency:
Time Complexity: O(n*2)
Auxiliary Space: O(1)
Boundary Cases: Insertion sort takes maximum time to sort if elements are sorted in reverse order. And it takes minimum time (Order of n) when elements are already sorted.
Algorithmic Paradigm: Incremental Approach
Sorting In Place: Yes
Stable: Yes
Online: Yes
Uses: Insertion sort is used when number of elements is small. It can also be useful when input array is almost sorted, only few elements are misplaced in complete big array.
