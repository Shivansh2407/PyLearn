Sequence in python is a data type that allows you to store and access your data sequentially.
The elements in the sequence are indexed where first element has index 0 then second 1, third 2 so and so forth.

There are 7 types of sequences in python the most important ones that we are going to concentrate on are:
1.  Lists
2.  Tuples
3.  Dictionary

There are certain things you can do with all sequence types. These operations include indexing, slicing, adding, multiplying, and checking for membership. In addition, Python has many built-in functions which makes handling these sequences easier and faster too.

Python Lists
A list in python is similar to an array in C++. They store the data sequentially with default index elements starting from zero. Every element can be accessed using it’s respective index.
A list is declared in square-brackets. The elements are written in a comma- seperated format.
Creating a list is as simple as putting different comma-separated values between square brackets.
Example −
list1 = ['physics', 'chemistry', 1997, 2000];
list2 = [1, 2, 3, 4, 5 ];
list3 = ["a", "b", "c", "d"]

Similar to string indices, list indices start at 0, and lists can be sliced, concatenated and so on.

Accessing Values in Lists
To access values in lists, we use the square brackets for slicing along with the index or indices to obtain value available at that index.
Example −
list1 = ['physics', 'chemistry', 1997, 2000];
list2 = [1, 2, 3, 4, 5, 6, 7 ];
print "list1[0]: ", list1[0]
print "list2[1:5]: ", list2[1:5]

Output:
list1[0]: physics
list2[1:5]: [2, 3, 4, 5]

Updating Lists
The most commonly used ways are−
1. Directly assigning values to the elements at the given index.(refer line 3 of the code below.)
2. Using append function to add elements at the end of the list. (refer line 6 of the code.)

Example-
list = [‘a’,’b’,0,1];
print (‘Value available at index 2 : ‘,list[2])
list[2] = 3;
print (‘New value available at index 2 : ‘,list[2])
print (‘Elements in the list are : ‘,list)
list.append(5)
print (‘The elements in the list now are:’,list)

Output:
Value available at index 2 :
0
New value available at index 2 :
3
Elements in the list are :
 ['a', 'b', 0, 1]
The elements in the list now are:
['a', 'b', 0, 1, 5]
