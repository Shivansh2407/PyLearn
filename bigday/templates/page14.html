Conditional statements in Python are used to make certain decisions. Suppose you have to go to a out today and you have to make a decision whether to take the umbrella or not to take.
So, A normal person would think:
            if:   it's Raining outside                    then: I will take the umbrella
            else: If it's not Raining outside       then: I will not take the umbrella
So these are desicions that follow some data processing and comparing. This task in python is performed by the conditional statements.
The conditional statements check the given statement with respect to the already present data and make proper desicions based on the result of the processing.

The commonly used keywords for conditional decision making are:
1. IF statement
2. IF - ELSE statement
3. IF -ELIF -ELSE statement
4. Nested IF-ELSE statements
5. Switch Case

IF statement :
IF statement is used to check a particular statement and based on the result either to execute a certain set of statements or not to execute them.

Example:
value=50
if value>0:
    print('The value is positive!')

                         

Output:
The value is positive!  



IF-ELSE statement:
IF-ELSE statement is used to check a particular statement and based on the result either to execute a certain set of statements or ELSE to execute some other set of statements.

Example:
age=int(input('Enter the age:'))
if age<1:
    print('The age cannot be less than 1!' )
else:
    print('Your age has been recorded . Thank You!')


Output:

Enter the age:0
The age cannot be less than 1!


IF-ELIF-ELSE statement:
IF- ELIF -ELSE statement is used to check a particular statement and based on the result either to execute a certain set of statements or to check another statement and to decide which set of statements to execute.
Example:

salary=50000
if salary>60000:
    print('The bonus given will be =',9000)
elif salary>30000:
    print('The bonus given will be =',6000)
else:
    print('The bonus given will be =',3000)



Output:
The bonus given will be =6000


Nested IF-ELSE statement:
Nested IF-ELSE statements are used to perform statements within statement checks. That is, one if statement in another and so on.

Example:
income=600000
if income>250000:

            if income>1000000:
