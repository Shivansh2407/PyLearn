                                                        other object.

Bitwise operators:
They are used to perform certain operations on binary digits.
Some of them are:

1. &   ( AND )                                      Sets each bit to 1                                                          x&y
                                                            if both bits are 1
2. |    ( OR )                                         Sets each bit to 1                                                          x|y
                                                            if one of two bits is 1
3. ^  (  XOR )                                      Sets each bit to 1                                                           x^y
                                                            if only one of two bits is 1
4. ~   ( NOT )                                     Inverts all the bits                                                          x~y
5. <<                                                   Takes two numbers, left shifts
                                                            the bits of the first operand,
                                                            the second operand decides
                                                            the number of places to shift
6. >>                                                   Takes two numbers, right shifts                                x>>y
                                                            the bits of the first operand,
                                                            the second operand decides
                                                            the number of places to shift
