In the dictionary every key is assigned values using semi-colon and two different keys are separated by comma.

Example-
dict = {'Python': 10, 'CPP': 7, 'Java': 9}
 An empty dictionary is declared using {}.

Accessing Values in Dictionary
To access dictionary elements, you can use the familiar square brackets along with the key to obtain its value just like the lists.

Example −
dict = {'Python': 10, 'CPP': 7, 'Java': 9}
print (‘dict['Python']: ‘, dict['Python'])
print (‘dict['CPP']: ‘, dict['CPP'])

Output:-
dict['Python']: 10
dict['CPP']: 7
print(dict['HTML'])
Traceback (most recent call last):

    print(dict['HTML'])

Updating Dictionary
You can modify the existing dictionary by using the keys and assigning them values.
dict = {'Python': 10, 'CPP': 7, 'Java': 9}
dict['CPP'] = 8; # update existing entry
Even if the key does not exist in the dictionary we can add it directly by assigning the value and writing the key directly.

Example-
dict['HTML'] = 8; # Add new entry
print (‘dict:’,dict)

Output:
{'Python': 10, 'CPP': 8, 'Java': 9, 'HTML': 8}

Delete Dictionary Elements
You can either remove individual dictionary elements or clear the entire contents of a dictionary. You can also delete entire dictionary in a single operation.
To explicitly remove an entire dictionary, just use the del statement.

Example −
dict = {'Python': 10, 'CPP': 8, 'Java': 9, 'HTML': 8}
del dict['Python’]; # remove entry with key 'Name'
print(dict)
dict.clear(); # remove all entries in dict
print(dict)

Output:
{'CPP': 7, 'Java': 9, 'HTML': 8}
{}

Properties of Dictionary Keys
There are no restrictions on the values that we assign to the keys. But the keys that we add must follow certain rules.

There are two important points to remember about dictionary keys −
(a) Duplication of keys is not allowed. In this case, the last assignment of that key is displayed.
dict = {'HTML': 7, 'Java': 9, 'HTML': 8}
print "dict['HTML']: ", dict['HTML']
Output:
dict['HTML']: 8
(b) The keys must be immutable. That means, the keys can be strings, numbers or tuples but they cannot be lists. As lists are mutable.
dict = {['CPP']: 7, 'Java': 9, 'HTML': 8}
Output:
Traceback (most recent call last):
  File "<pyshell#9>", line 1, in <module>
    dict = {['CPP']: 7, 'Java': 9, 'HTML': 8}
TypeError: unhashable type: 'list'

Built-in Dictionary Functions & Methods
Python includes the following dictionary functions −

1 . Compare                                                                           cmp(dict1, dict2)
Compares elements of both dict.
2 . Length                                                                              len(dict)
Gives the total length of the dictionary. This would be equal to the number of items in the dictionary.
3 . Dict2Str                                                                           str(dict)
Produces a printable string representation of a dictionary
4 TypeOfVariable                                                                type(variable)
Returns the type of the passed variable. If passed variable is dictionary, then it would return a dictionary type.

Python includes following dictionary methods −

1 . Clear                                                                                 dict.clear()
Removes all elements of dictionary dict

2 . Copy                                                                                 dict.copy()
Returns a shallow copy of dictionary dict

3 . Dict2List                                                                          dict.fromkeys()
Create a new dictionary with keys from seq and values set to value.

4 . GetKey's Value                                                               dict.get(key, default=None)
Returns the value of the key if True ,else returns default value

5 . PresenceOfKey                                                              dict.has_key(key)
Returns true if key in dictionary dict, false otherwise

6 .Items                                                                                 dict.items()
Returns a list of dict's (key, value) tuple pairs

7 . Keys                                                                                  dict.keys()
Returns list of dictionary dict's keys

8 .SetDefault                                                                       dict.setdefault(key, default=None)
Similar to get(), but will set dict[key]=default if key is not already in dict

9 . Update                                                                             dict.update(dict2)
Adds dictionary dict2's key-values pairs to dict

10 . Values                                                                            dict.values()
Returns list of dictionary dict's values
