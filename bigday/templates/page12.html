A character is a symbol which lies in the ASCII code. For example, the English language has 26 characters.
A strings are a sequence of characters. These characters are all stored in the memory together and collectively make some sense.
Every character has a unique ASCII code that is used to store and retrieve it by the computer as it only understands the language of 1s and 0s.

How to create a string in Python?
Strings can be created by enclosing characters inside a single quote or double quotes. Even triple quotes can be used in Python but generally used to represent multiline strings and docstrings.
Example-
my_string = 'Hello'
print(my_string)
my_string = "Hello"
print(my_string)
my_string = '''Hello'''
print(my_string)
# triple quotes string can extend multiple lines
my_string = """Hello, welcome to
           the world of Python"""
print(my_string)

Output:
Hello
Hello
Hello
Hello, welcome to
           the world of Python


Accessing characters in string:
We can access individual characters using indexing and a range of characters using slicing. Index starts from 0. Trying to access a character out of index range will raise an IndexError. The index must be an integer. We can't use float or other types, this will result into TypeError.
Python allows negative indexing for its sequences.
The index of -1 refers to the last item, -2 to the second last item and so on. We can access a range of items in a string by using the slicing operator (colon).

Example:
string = ‘Hello World’
print('string= ', string)
#first character
print('string[0] = ', string[0])
#last character
print('string[-1] = ', string[-1])
#slicing 2nd to 5th character
print('string[1:5] = ', string[1:5])
#slicing 6th to 2nd last character
print('string[5:-2] = ', string[5:-2])
If we try to access index out of the range or use decimal number, we will get errors.
# index must be in range
>>> string[20]
...
IndexError: string index out of range
# index must be an integer
>>> string[1.5]
...
TypeError: string indices must be integers


Updating and deleting in string:
Strings are immutable. This means that elements of a string cannot be changed once it has been assigned. We can simply reassign different strings to the same name.

Example(in python shell):
>>>string = 'Hello World'
>>> string[5] = 'a'
Traceback (most recent call last):
  File "<pyshell#12>", line 1, in <module>
    string[5]='a'
TypeError: 'str' object does not support item assignment
>>> string = 'Python'
>>> string
'Python'

We cannot delete or remove characters from a string. But deleting the string entirely is possible using the keyword del.

Example:
>>> del string[0]
Traceback (most recent call last):
  File "<pyshell#13>", line 1, in <module>
    del string[0]
TypeError: 'str' object doesn't support item deletion
>>> del string
>>> string
...
NameError: name 'string' is not defined

Note:The NameError occurs because the entire string has been deleted and cannot be accessed in the program.


Python String Operations
There are many operations that can be performed with string which makes it one of the most used datatypes in Python.

Concatenation of Two or More Strings
Joining of two or more strings into a single one is called concatenation.
The + operator is used for concatenation in python. Simply writing two string literals together also concatenates them.
The * operator can be used to repeat the string for a given number of times.

Example:
str1 = 'Hello'
str2 ='World!'
# using +
print('str1 + str2 = ', str1 + str2)
# using *
print('str1 * 3 =', str1 * 3)

Output:
str1 + str2 = HelloWorld!
str1 * 3 = HelloHelloHello

Writing two string literals together also concatenates them like + operator.
If we want to concatenate strings in different lines, we need to use parentheses.

Example:
>>> # two string literals together
>>> 'Hello ''World!'
'Hello World!'
>>> # using parentheses
>>> s = ('Hello '
              'World')
>>> s
'Hello World'


Iterating Through String
We can use FOR loop to iterate through the elements of a string just like the elements of a list.
We mostly use the len(str) keyword to know the length of the string.

Example:
Here is a simple program to count the number of times 'a' occurs in the string.

total=0
string='Keep Calm and Learn to Code'
length=len(string)
for i in range(0,length):
    if string[ i ]=='a':
        total+=1
print('The number of times "a" occurs is:' , total)
        
Output:
The number of times "a" occurs is: 3


String Membership Test
We can test if a sub string exists within a string or not, using the keyword in.
It can also help us to iterate through a given string.
>>> 'a' in 'program'
True
>>> 'at' not in 'battle'
False

We can use in keyword to access each element of the string sequentially when used with for loop. The same above code can also be written using in keyword.

Example: 
total=0
string='Keep Calm and Learn to Code'
for i in string:
    if i=='a':
        total+=1
print('The number of times "a" occurs is:' , total)

Output:
The number of times "a" occurs is: 3

Note:The program becomes much more simpler and lesser number of variables need to be used making the code more efficient.

Built-in functions to Work with Python

Various built-in functions that work with sequence, works with string as well.
Some of the commonly used ones are enumerate() and len().

The enumerate() function returns an enumerate object. It contains the index and value of all the items in the string as pairs. This can be useful for iteration.
Similarly, len() returns the length (number of characters) of the string.

Example:
str = 'cold'
# enumerate()
list_enumerate = list(enumerate(str))
print('list(enumerate(str) = ', list_enumerate)
#character count
print('len(str) = ', len(str))

Output:
list(enumerate(str) = [(0, 'c'), (1, 'o'), (2, 'l'), (3, 'd')]
len(str) = 4
