A variable is a reserved memory location which contains some value. Variables can simply be alphabets or words or any sort of such thing. The value the variable contains can be of any type: string, float or integer. That is where the Data Types come into existence.

 The difference between a variable and a constant is :
The variable can be changed and assigned values in the course of the program execution. But the value of the constant always remains the same and cannot be modified throughout the execution of the code.

Rules to be followed to name variables:
1. The variable name must start from either an alphabet or an underscore only.
2. The variable name (otherthan the first letter) can contain numbers and alphabets but no special symbols are allowed.
3.  No space must be given in a variable name.
4. Variable names are case sensitive. So, uppercase letters are different than lower case letters.
5. No two variables must have the same name as it might lead to discrepancy. 
6. You cannot use a python keyword(reserved word) as a variable.

Correct variables :  'Name'     'Age'   ,   'age'   ,  ' _date'   , 'Phone_number1'  ,   'Phone_number2'
Incorrect variables :  'int'   ,  'print'   ,  '1phone number' ,  '2Phone_number'  ,  ' Name@1'


Assigning values to a variable:
We can declare variables in python without actually worrying about their data types. We can directly use a '=' sign to assign values to the variable. This will also declare the variable.
We can only access a variable if it has been declared previously in the program.
We can also assign the values from the user using input() function. The default input type is a string in python.
To input integer data we need to specify int()  while taking the input.
To input decimal data we need to specify float() while taking the input.

Example (written in python shell):

>>> country='India'                                                                       # String value assigned
>>> print(country)
India

>>> population=100                                                             #Integer value assigned
>>> print(population)
100

>>> Rate=0.75                                                                            #Decimal value assigned
>>> print(Rate)                                                                 
0.75

>>> Age=int(input('Enter your age: '))                                  # Integer value is specified to be inputed

>>> Rate=float(input('Enter the rate:'))                                    #Float value is expected from the user

>>> print(Rate)

Updating the value of a variable:
We can directly assign the values to the variables using '=' operator any number of times. The last value assigned will be stored in the variable.

Example(written in python IDLE):

Salary=30000
print('Initial Salary is: ',Salary)
Salary=30000+5000
print('Updated Salary is: ',Salary)

Output:
Initial Salary is: 30000
Updated Salary is: 35000
